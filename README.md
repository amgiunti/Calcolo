# Calcolo

## Scopo dello script:
Operazione di addizione in maniera da avere comne risultato la somma di due numeri interi, in caso di float number, non verranno considerati i numeri dopo la virgola.

## Prerequisiti:
PHP 8.2


## Comandi di esecuzione con esempi:

1. php calc.php  
2. php calc.php 8
3. php calc.php -1
4. php calc.php 2,7

## Output atteso:

1. 1
2. 9 
3. 0
4. 3
