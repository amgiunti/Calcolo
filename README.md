# Calcolo

## Scopo dello script:
Operazione di addizione del numero 3 in maniera da avere comne risultato la somma dei due numeri interi. In caso di float number, non verranno considerati i numeri dopo la virgola.

## Prerequisiti:
PHP 8.2


## Comandi di esecuzione con esempi:

1. php calc.php  
2. php calc.php 8
3. php calc.php -4
4. php calc.php 2,7

## Output atteso:

1. 3
2. 11
3. -1
4. 5
